Операционная система: Windows 10 
Среда разработки: Visual Studio 2022
Компилятор: GCC
Библиотеки: <time.h>, <stdlib.h>
Сборка:
1) Откройте Visual Studio и создайте новый проект. Выберите тип проекта "CLR" -> "Пустой проект CLR".
2) Выберите целевую платформу Windows
3) Открыть файл "Game.h" и заменить его содержимое представленным кодом.
4) Открыть файл "Game.cpp" и заменить его содержимое представленным кодом.
